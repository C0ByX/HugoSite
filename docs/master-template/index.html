    Мастер-шаблон для маркетинговых рассылок #

    Этот мастер-шаблон предназначен для маркетинговых рассылок — регулярных и триггерных.

    В нашем бизнесе пока нет транзакционных рассылок. При необходимости мы сделаем для них отдельный мастер-шаблон.

    Мастер-шаблон находится здесь. Для доступа вам потребуется логин и пароль от Stripo.
